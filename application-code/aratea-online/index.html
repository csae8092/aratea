Aratea Online

A collection of digitized ressources of translations and excerpted versions based on the didactic poem Phaenomena of Aratus of Soli

The Aratea Online Project focuses on the manuscript transmission of the Aratea texts (translations and excerpted versions based on the didactic poem Phaenomena of Aratus of Soli). The project has two main objectives. First, it aims to describe all pre-13th c. manuscripts of the texts according to TEI. Thus, the user is able to search within the collection using queries.
